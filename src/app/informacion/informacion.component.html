SuperHero API

SuperHero API es una potente herramienta que te permite acceder a información detallada sobre cientos de superhéroes y villanos de distintos universos. Obtén datos como nombres reales, habilidades, poderes, apariciones en cómics, biografías y más, todo a través de una API fácil de usar. Ideal para desarrolladores, creadores de contenido y entusiastas de los superhéroes, esta API te ayuda a integrar información dinámica en tus proyectos, desde aplicaciones hasta páginas web y bots interactivos.

Al no tener la web en un hosting la api no te deja utilizar llamadas desde localhost por lo que me tocó poner una url delante de la llamada que su funcion es hacerlo pasar como si fuera desde un hosting.

Para hacer las llamadas primero tines que entrar a la web del programa para activarlo y así funcione la llamada a la api
